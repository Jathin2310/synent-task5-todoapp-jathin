# To-Do List Web App

## Overview
It is a modern and responsive To-Do List Web Application built using HTML, CSS, and JavaScript. It helps users organize and manage daily tasks efficiently through a clean and user-friendly interface.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All, Active, Done)
- Task counter and progress tracking
- Persistent storage using localStorage
- Responsive and minimalist design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- localStorage API

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
